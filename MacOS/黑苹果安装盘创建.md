# OpCore-Simplify

官网：https://github.com/lzhoang2801/OpCore-Simplify

运行OpCore-Simplify.bat生成设备报告和EFI

注：无线网卡选择itlwm，更稳定但是不支持Apple的控制中心加入热点，需要启动系统后安装HeliPort

HeliPort官网：https://github.com/OpenIntelWireless/HeliPort

# USBToolBox

官网：https://github.com/USBToolBox

下载发布版，运行windows.exe生成UTBMap.kext文件夹，将UTBMap.kext复制到生成EFI文件夹OC\Kexts\目录下，并删除UTBDefault.kext文件夹

# ProperTree

官网：https://github.com/corpnewt/ProperTree

运行ProperTree.bat

* CTRL+O，打开EFI\OC\config.plist
* CTRL+R，选择EFI\OC目录
* CTRL+S，保存

# 下载恢复镜像

进入目录：OpenCorePkg\Utilities\macrecovery，执行下载命令

不同版本下载的命令列表：https://dortania.github.io/OpenCore-Install-Guide/installer-guide/mac-install-recovery.html

```shell
# 例如Sequoia (15)
python3 ./macrecovery.py -b Mac-7BA5B2D9E42DDD94 -m 00000000000000000 download
```

下载后的恢复镜像放在com.apple.recovery.boot文件夹中

# 制作启动U盘

* U盘格式化为非可引导、FAT32格式
* 复制EFI文件夹到U盘根目录
* 复制com.apple.recovery.boot文件夹U盘根目录
