# 监测nvme固态硬盘温度和健康状况

```shell
#
* 
# 1. 安装nvme-cli
sudo apt-get install nvme-cli

# 2. 查看nvme硬盘列表
sudo nvme list

# 3. 查看nvme硬盘详细信息
sudo nvme smart-log /dev/nvme0
sudo watch -n 1 nvme smart-log /dev/nvme0
sudo nvme smart-log /dev/nvme0 | grep "^temperature"
```

# 查看传感器温度

```shell
# 安装
sudo apt install lm-sensors -y
yes | sudo sensors-detect

# 运行
sensors
```
