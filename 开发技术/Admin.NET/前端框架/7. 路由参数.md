## [普通路由](https://adminnet.top/frontendbook/introduce.html#%E6%99%AE%E9%80%9A%E8%B7%AF%E7%94%B1)

代码位置：[/@/views/params/common](https://gitee.com/lyt-top/vue-next-admin/blob/master/src/views/params)

### [1\. 配置路由](https://adminnet.top/frontendbook/introduce.html#_1-%E9%85%8D%E7%BD%AE%E8%B7%AF%E7%94%B1)

```
{
  path: '/params',
  name: 'paramsIndex',
  component: () => import('/@/layout/routerView/parent.vue'),
  redirect: '/params/common',
  ...,
  children: [
    {
      path: '/params/common',
      name: 'paramsCommon',
      component: () => import('/@/views/params/common/index.vue'),
      ...,
    },
    {
      path: '/params/common/details',
      name: 'paramsCommonDetails',
      component: () => import('/@/views/params/common/details.vue'),
      ...,
    },
  ]
}
```

### [2\. 页面跳转](https://adminnet.top/frontendbook/introduce.html#_2-%E9%A1%B5%E9%9D%A2%E8%B7%B3%E8%BD%AC)

`router.push`，传参使用 `query`

```
router.push({
  path: "/params/common/details",
  query: { id: Math.random(), name: "vue-next-admin" },
});
```

### [3\. 参数获取](https://adminnet.top/frontendbook/introduce.html#_3-%E5%8F%82%E6%95%B0%E8%8E%B7%E5%8F%96)

您可能需要了解 [动态路由匹配](https://next.router.vuejs.org/zh/guide/essentials/dynamic-matching.html)。`route.query`

```
<script setup lang="ts" name="paramsCommonDetails">
  import { onMounted } from "vue";
  import { useRoute } from "vue-router";

  // 定义变量内容
  const route = useRoute();

  // 页面加载时
  onMounted(() => {
    // 参数：query: { "id": "111", "name": "vue-next-admin" }
    console.log(route.query);
  });
</script>
```

## [动态路由](https://adminnet.top/frontendbook/introduce.html#%E5%8A%A8%E6%80%81%E8%B7%AF%E7%94%B1)

### [1\. 配置路由（动态）](https://adminnet.top/frontendbook/introduce.html#_1-%E9%85%8D%E7%BD%AE%E8%B7%AF%E7%94%B1-%E5%8A%A8%E6%80%81)

注意 `/params/dynamic/details/:t/:id` 中的 `/:t/:id`，您可能需要了解 [路由的匹配语法](https://next.router.vuejs.org/zh/guide/essentials/route-matching-syntax.html)

```
{
  path: '/params',
  name: 'paramsIndex',
  component: () => import('/@/layout/routerView/parent.vue'),
  redirect: '/params/dynamic',
  ...,
  children: [
    {
      path: '/params/dynamic',
      name: 'paramsDynamic',
      component: () => import('/@/views/params/dynamic/index.vue'),
      ...,
    },
    {
      path: '/params/dynamic/details/:t/:id',
      name: 'paramsDynamicDetails',
      component: () => import('/@/views/params/dynamic/details.vue'),
      ...,
    },
  ]
}
```

### [2\. 页面跳转（动态）](https://adminnet.top/frontendbook/introduce.html#_2-%E9%A1%B5%E9%9D%A2%E8%B7%B3%E8%BD%AC-%E5%8A%A8%E6%80%81)

注意这里用 `name` 跳转，参数用 `params`

```
// name 值为路由中的 name
router.push({
  name: "paramsDynamicDetails",
  params: {
    t: "vue-next-admin",
    id: Math.random(),
  },
});
```

### [3\. 参数获取（动态）](https://adminnet.top/frontendbook/introduce.html#_3-%E5%8F%82%E6%95%B0%E8%8E%B7%E5%8F%96-%E5%8A%A8%E6%80%81)

您可能需要了解 [动态路由匹配](https://next.router.vuejs.org/zh/guide/essentials/dynamic-matching.html)。`route.params`

```
<script setup lang="ts" name="paramsDynamicDetails">
  import { onMounted } from "vue";
  import { useRoute } from "vue-router";

  // 定义变量内容
  const route = useRoute();

  // 页面加载时
  onMounted(() => {
    // 参数：params: { "t": "vue-next-admin", "id": "111" }
    console.log(route.params);
  });
</script>
```
