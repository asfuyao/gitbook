# 数据库中是空格网页输出是&nbsp;

在数据库中看起来是空格，实际输出到html时变成了&nbsp;，这是因为这个空格是特殊的空白字符，解决的办法：

```sql
REPLACE(Your_Column, NCHAR(160), ' ')
```

```sql
use master
```
